# evropa.eu.org

Research platform on Georgia's European integration process and the impact of disinformation.

## SEO & Social Media Features

This website includes comprehensive SEO optimization and social media integration:

### Search Engine Optimization
- **Meta Tags**: Description, keywords, author, and language tags on all pages
- **Canonical URLs**: Prevent duplicate content issues
- **Robots.txt**: Guide search engine crawlers
- **Sitemap.xml**: Complete site structure for search engines
- **Structured Data**: JSON-LD schema markup (WebSite, Article, Organization, BreadcrumbList)

### Social Media Integration
- **Open Graph (OG) Protocol**: Optimized for Facebook, LinkedIn, and other platforms
  - Page-specific titles and descriptions
  - Image references (og-image.jpg, og-image-disinfo.jpg, etc.)
  - Locale set to Georgian (ka_GE)
- **Twitter Cards**: Large image cards for better Twitter appearance
- **Theme Color**: Brand color (#003399) for mobile browsers

### Progressive Web App Support
- **Web App Manifest** (manifest.json): Installable web app configuration
- **Favicon Support**: Multiple sizes for different devices
- **Apple Touch Icons**: iOS home screen support

## Pages

- **მთავარი** (Home): Overview of the European integration research
- **დეზინფორმაცია** (Disinformation): Analysis of disinformation in digital media
- **მედიის ისტორია** (Media History): Evolution of Georgian media from 1925 to digital era
- **საზოგადოება** (Society): Public opinion research and media literacy statistics

## Note for Webmaster

To complete the SEO setup:
1. Add Open Graph images (1200x630px recommended):
   - `/og-image.jpg` - Main site image
   - `/og-image-disinfo.jpg` - Disinformation page
   - `/og-image-history.jpg` - Media history page
   - `/og-image-society.jpg` - Society page
2. Add favicon files:
   - `/favicon.ico`
   - `/favicon-16x16.png`
   - `/favicon-32x32.png`
   - `/apple-touch-icon.png` (180x180px)
3. Update sitemap.xml lastmod dates when content changes
4. Submit sitemap.xml to Google Search Console and Bing Webmaster Tools
